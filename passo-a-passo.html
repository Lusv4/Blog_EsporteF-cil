Passo a Passo

passo a passo para a realização desse teste e criação da página no GitHub Pages.

Repositório

Crie um repositório no GitHub

Settings

Clica em Settings

Pages

Clique em Pages

Actions

Mude para a opção de cima "GitHub Actions"

Configure

Clique em configure no card "Static HTML"

Commit Changes

Clique em Commit Changes. Depois volte para a pagina "pages" no mesmo repositório

Visit Site

Você pode acessar seu site e copiar o seu endereço do link.

PageSpeed Insights

Acesse o site PageSpeed Insights e coloque o endereço do link para avaliar o seu site.
